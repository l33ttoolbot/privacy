Datenschutz AGB

Datenschutzerklärung

Kurzfassung: Beemaster speichert alle Daten lokal auf deinem Gerät. Persönliche Daten werden verschlüsselt. Es werden keine Werbung oder Marketing-Daten gesammelt. Deine Bienenstock-Daten bleiben bei dir.

1. Verantwortlicher

Verantwortlich für die Datenverarbeitung:

  • Entwickler: Mike Miller
  • E-Mail: privacy@beemaster.app
  • Telegram: @mightmikemiller

2. Welche Daten werden erhoben?

📱 Erforderliche App-Daten

Daten, die für die Grundfunktionen der App benötigt werden:

Datentyp Zweck Speicherung
Bienenstock-Daten Verwaltung deiner Imkerei Lokal auf Gerät
Inspektions-Notizen Dokumentation Lokal auf Gerät
GPS-Standorte Kartenansicht der Bienenstöcke Lokal auf Gerät
Erinnerungen Benachrichtigungen Lokal auf Gerät

👤 Optionale Persönliche Daten

Für das Imker-Profil kannst du optional angeben:

  • Name / Imkerei-Name
  • E-Mail-Adresse
  • Telefonnummer
  • Adresse / Standort

✓ Verschlüsselung: Persönliche Daten werden mit Android Keystore verschlüsselt gespeichert.

📸 Medien

  • Fotos: Bienenstock-Bilder (optional, lokal gespeichert)
  • Sprachnotizen: Audio-Aufnahmen (optional, lokal gespeichert)

📊 Automatisch erhobene Daten

Datentyp Zweck An Dritte?
Crash-Logs App-Stabilität verbessern Firebase Crashlytics
Nutzungsstatistiken App-Verbesserung Firebase Analytics
Geräte-ID Fehlerdiagnose Firebase

⚠️ Firebase-Datenschutz: Crash-Logs und anonyme Nutzungsdaten werden an Google Firebase übertragen. Weitere Informationen: firebase.google.com/support/privacy

3. Daten NICHT erhoben

Beemaster sammelt nicht:

  • ❌ Keine Werbe-ID
  • ❌ Keine Marketing-Daten
  • ❌ Keine Kontakte ohne Einwilligung
  • ❌ Keine SMS oder Anrufe
  • ❌ Keine Browser-Historie
  • ❌ Keine Gesundheitsdaten
  • ❌ Keine Finanzdaten
  • ❌ Keine Social-Media-Profile

4. Berechtigungen

Berechtigung Verwendung
Standort (GPS) Bienenstöcke auf Karte anzeigen
Kamera Fotos von Bienenstöcken
Mikrofon Sprachnotizen
Benachrichtigungen Erinnerungen für Imker-Aufgaben
Speicher Kartendaten (OSMDroid)

5. Datenspeicherung

Lokal auf deinem Gerät

  • Room Database: Alle Bienenstock-Daten
  • DataStore: App-Einstellungen
  • Android Keystore: Verschlüsselte sensible Daten

Keine Cloud (Basis-Version)

✓ Deine Daten bleiben bei dir: In der kostenlosen Version werden keine Daten auf Server gespeichert. Alle Daten bleiben lokal auf deinem Gerät.

6. Drittanbieter

OpenAI API (optional)

  • Zweck: KI-basierte Imker-Assistenz (Premium)

Du hast das Recht auf:

  • Auskunft: Welche Daten über dich gespeichert sind

8. Kinder- und Jugendschutz

Diese App ist nicht für Kinder unter 13 Jahren bestimmt. Es werden keine gezielten Daten von Kindern erhoben.

9. Datensicherheit

  • ✅ Verschlüsselte Datenübertragung (HTTPS/TLS)
  • ✅ Verschlüsselte Speicherung (Android Keystore)
  • ✅ Keine Cloud-Speicherung (lokale Daten)
  • ✅ Regelmäßige Sicherheits-Updates
